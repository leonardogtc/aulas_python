# Dicionários

## O que são dicionários em python 3.12?

- Coleções de pares **chave:valor** são chamadas de dicionários
- São capazes de armazenar uma quantidade quase ilimitada de informações 
- As informações são armazenadas em pares chave:valor (isso quer dizer que cada chave é conectada a um valor correspondente)
- Entender os dicionários permite modelar uma diversidade de objetos do mundo real de modo mais preciso.
- Em Python, um dicionário é representado entre chaves, {}, com uma série de pares chave-valor entre elas
- O valor de uma chave pode ser um número, uma string, uma lista ou até mesmo outro dicionário.
- Podemos usar qualquer objeto que possa ser criado em Python como valor de um dicionário
